# 의석이의 세로로 말해요~



## code

```python
T = int(input())
for test_case in range(1,T+1):
    S = [list(input()) for _ in range(5)] # 입력
    result = '' # 결과 받을 빈 문자열 생성

    cnt = 0
    while cnt < 5: # 5개 비면 stop!
        cnt = 0
        for i in S: # 반복
            if i:
                s = i.pop(0) # 맨 앞 지우고
                result += str(s) # 빈문자 열에 넣기 
            else: # 비었으면 
                cnt += 1 # 카운트 +1

    print(f'#{test_case} {result}') 

```
